- Built Vue 3 based frontend with JavaScript from the ground up
- Developer Next.js based frontend with TypeScript
- Designed modular Python Flask APIs supporting 30+ integrations (Salesforce, HubSpot, Snowflake, BigQuery)
- Managed PostgreSQL databases and CI/CD pipelines on Google Cloud
- Scaled platform to serve 105 paying customers

Full-stack Spotify enhancement tool with playlist management and music analytics
- Built with React, TypeScript, and Tailwind CSS
- Backend powered by Express.js with Zod validation
- CI/CD with Docker and GitHub Actions, deployed on Fly.io
